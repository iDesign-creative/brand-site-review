iDesign's Creative + Marketing Site Reviewer

1 Add the button (one time)

Easiest: drag this button up to your bookmarks bar. (Show the bar with ⌘⇧B on Mac / Ctrl⇧B on Windows.)

Comment on iDesign

or paste it
If dragging won't work, create a new bookmark and paste this as its URL / Address:

  1. Chrome/Edge: ⋮ menu → Bookmarks → Bookmark manager → ⋮ → Add new bookmark. Name it "Comment on iDesign", paste the code into URL, Save.
  2. Safari: Bookmarks → Add Bookmark on any page, then Edit Bookmarks and replace its address with the pasted code.
The button works on any iDesign site — each site's comments are kept separate automatically.

2 Use it on the site you're reviewing

  1. Open the site you were asked to review.
  2. Click the Comment on iDesign bookmark. A panel opens top-right.
  3. Enter your email → + Add comment → click any text or section → type your note.
  4. Reply to others and mark items resolved. Everyone's comments sync live.
